# GenAI Bootcamp – Use Case Implementations

## Overview
This repository contains different use case implementations developed as part of the **GenAI Bootcamp** assignment. Each folder represents a specific use case demonstrating the application of Generative AI techniques and tools.

The goal of these implementations is to explore practical applications of Large Language Models (LLMs), prompt engineering, and AI-powered pipelines.

## Purpose
This repository serves as a collection of bootcamp exercises designed to demonstrate:

- Practical usage of Generative AI tools
- Implementation of different AI pipelines
- Experimentation with open-source LLMs
- Hands-on learning through real-world use cases

Each use case is implemented independently and may use different tools, models, or approaches depending on the problem being solved.
